A Passion for Analysis

Now that the children are older, it makes more sense to have the teacher teach the children and the scientist go to work. My thesis work developed Rietveld (least-squares) analysis procedures to determine preferred orientations (texture) of mineral crystals in rocks. I worked with both image and text data taken from several different sources. I loved taking in large amounts of data in often difficult formats and turning it into useful information. It might take months of analysis and modeling to create a single graphic, but that graphic could show the strain history across an entire region.

Data Science Projects

Identifying Fraud from Enron Email

Using the Enron Corpus and public financial data, compared Random Forest and AdaBoost at identifying Persons of Interest. I also got an introduction to text learning by stemming and vectorizing email text and training a classifier to select between two email authors.

Interactive Crime Map of London Boroughs

Visualized data from the London Data Store as an app in R and deployed the app on the Shiny server

London Crime Map Presentation

Created a brief HTML5 slide deck in R describing the London crime map app.

Classifying Workout Quality from Personal Accelerometer

Trained and tested a random forest algorithm on accelerometer records taken from subjects performing bicep curls to identify problems in technique.

Exploratory Data Analysis of Prosper Loans

Project 4 of the Udacity Data Analyst Nanodegree. Univariate, bivariate, and multivariate analysis performed in a stream-of-consciousness style Rmarkdown file.

Linear Regression Model for Mtcars

Developed a linear regression model for fuel economy using R's mtcars dataset. Rather than rely soley on R-squared and variance inflation factors, I did some research on cars to develop a more physical model.

Statistical Comparisons using T-Tests and Confidence Intervals

Quick comparisons using t-tests and confidence intervals with R's ToothGrowth dataset.

Statistical Inference Simulation Exercise

An exercise to illustrate the Central Limit Theorem, which states that the distribution of a sample statistic is normal with mean equal to the population statistic it estimates and variance equal to the square of the standard error (population variance divided by sample size). Uses R to perform 1000 simulations of drawing 40 random numbers from an exponential distribution with lambda = 0.2. Calculates a sample mean for each simulation and demonstrates that they are normally distributed.

Exploratory Analysis of NOAA Storm Database

The second project for Reproducible Research was a chance to play with the National Weather Service's Storm Database. The question was simply which types of weather event cause the most damage and risk to human life. The analysis was complicated by dirty data. There were almost 500 unique entries for the type of weather event.

Descriptive Analysis of Personal Activity Data, Oct-Nov 2012

This project was the first peer assessment for Reproducible Research, the fifth course in the Johns Hopkins Data Science Specialization. This assignment makes use of data from a personal activity monitoring device. This device collects data at 5 minute intervals through out the day. The data consists of two months of data from an anonymous individual collected during the months of October and November, 2012 and include the number of steps taken in 5 minute intervals each day.

Wrangling OpenStreetMaps data into MongoDB

For the Udacity Data Wrangling course, I downloaded the OpenStreetMaps data for my area, cleaned some aspects of the data in python and converted the XML data into JSON format. I inserted the JSON data into a MongoDB database and did some exploration of the user-entered information.

Statistical analysis of rain's effect on NYC subway ridership

This project for Udacity's Intro to Data Science course combined SQL queries of WeatherUnderground data with csv files from the New York Transit Authority. The analysis was done in python using pandas and pandasql.

Exploratory Analysis of the EPA's National Emissions Inventory Data 1999-2008

This project explored fine particulate matter emissions in the US using the R package dplyr for analysis and the different graphics packages in R: base, lattice, and ggplot2. It was the second project for Exploratory Data Analysis.

Creating a tidy dataset of smartphone accelerometer readings

Compiled text files into a single dataframe using dplyr package for R and output a tidy dataset ready for analysis. Project for Getting and Cleaning Data course of Johns Hopkins data science specialization.

Exploration of Power Consumption Data

Examined how household energy usage varied over a 2-day period in February, 2007, using the base plotting system in R.
